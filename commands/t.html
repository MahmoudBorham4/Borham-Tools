The T command creates a professional coordinate table with automatic code detection from nearby text labels, perfect for survey reports and point documentation.

NEW: Code Column Feature

  • Automatic code extraction from nearest text labels
  • Smart search radius (within 5 units)
  • Supports TEXT and MTEXT objects
  • Displays "N/A" if no text found
  • Perfect for labeled points (BM-01, TP-05, etc.)

üîç Smart Code Detection

How It Works:

  1. For each selected point, searches within 5-unit radius
  2. Looks for TEXT and MTEXT objects
  3. Uses nearest text content as code
  4. If no text found ‚Üí Shows "N/A"
  5. Perfect for survey points with labels like:
    • BM-01 (Benchmark)
    • TP-05 (Turning Point)
    • COL-A (Column)
    • MH-12 (Manhole)

Search Parameters:

  • Search Radius: 5 drawing units
  • Text Types: TEXT, MTEXT
  • Priority: Nearest text first
  • Fallback: "N/A" if no text found
