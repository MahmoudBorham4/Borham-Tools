PI

Point Import

üìã Description

Imports points from TXT, CSV, or PNT files with support for multiple coordinate formats. Features a professional dialog box for complete control over import options.

üìä Supported File Formats

  • Format 1: P X Y Z C (Point Name, X, Y, Z, Code)
  • Format 2: P Y X Z C (Point Name, Y, X, Z, Code)
  • Format 3: X Y Z C (Auto-numbered points)
  • Format 4: Y X Z C (Auto-numbered points)

Supports: Comma-separated (CSV) or Space-separated (TXT) files

‚ú® Import Options

  • Import Points: Create point entities (2D or 3D)
  • Import Names: Add point name labels
  • Import Codes: Add point code labels
  • Import Elevation: Add elevation text
  • Connect Polyline: Connect points with 2D or 3D polyline

‚öôÔ∏è Point Types

  • 2D Points (Z=0): All points placed at Z=0
  • 3D Points (Elevated): Points placed at their actual Z elevation

üîó Polyline Types

  • 2D Polyline: Connects points in XY plane (Z=0)
  • 3D Polyline: Connects points with their Z elevations

üé® Layer Management

  • As-built: Point entities (Color: Red)
  • As-Built-Point-Name: Point name labels (Color: Yellow)
  • As-built-Codes: Point codes (Color: Green)
  • As-built-Elevation: Elevation text (Color: Cyan)
  • As-built-Polyline: Connected polyline (Color: Magenta)

üìù Text Settings

  • Text Height: Configurable (default: 0.2)
  • Text Rotation: Adjustable in degrees (default: 0)
  • Smart Positioning: Labels placed around points automatically

üí° How to Use

  1. Type PI and press Enter
  2. Configure import options in the dialog box:
    • Select what to import (Points, Names, Codes, Elevation)
    • Choose point type (2D or 3D)
    • Select coordinate format
    • Set text height and rotation
    • Enable polyline connection if needed
  3. Click OK
  4. Browse and select your points file (TXT/CSV/PNT)
  5. Points and labels will be imported automatically
  6. View zooms to fit all imported points
