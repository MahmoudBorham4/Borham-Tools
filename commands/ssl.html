Select Similar Objects with Intelligent Property Matching

üìñ Command Overview

The SSL command intelligently selects objects similar to reference objects throughout the entire drawing using advanced property matching algorithms.

Key Capabilities:

  • Multi-criteria matching by object type
  • Multiple reference object support
  • Smart property caching for lightning speed
  • Automatic layer management (unlocks/frozen layers)
  • Performance timing display
  • Duplicate property filtering

üéØ Matching Criteria by Object Type

  • LINES / POLYLINES / ARCS / SPLINES / ELLIPSES: Length
  • CIRCLES: Radius
  • HATCH: Pattern
  • TEXT / MTEXT: Style & Height
  • INSERTS (Blocks): Block Name & Scale
  • DIMENSIONS: Dimension Style
  • OTHER OBJECTS: Type, Layer & Color

üöÄ How to Use

To select similar objects in AutoCAD:

  1. Type SSL in the command line and press Enter
  2. Select one or more reference objects
  3. Press Enter to confirm selection
  4. Watch as similar objects are selected automatically throughout the drawing
  5. Use standard AutoCAD commands to modify the selected objects

Command Line Example:

Command: SSL
Select reference objects: [select objects]
Found 24 similar objects. Time: 0.15s
Objects selected successfully!
