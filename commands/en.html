EN

Easting (X) and Northing (Y) Coordinate Annotation

üìñ Command Overview

The EN command creates professional MLeader annotations displaying Easting (X) and Northing (Y) coordinates with configurable scale and angle settings.

Key Features:

  • Professional MLeader formatting
  • Configurable scale (global variable: en_scale)
  • Adjustable angle (global variable: en_angle)
  • Automatic layer creation ("COORDINATES" - Color: Red)
  • Continuous operation mode
  • Interactive adjustments during command

üöÄ How to Use

To annotate coordinates in AutoCAD:

  1. Type EN in the command line and press Enter
  2. Click on the point you want to annotate
  3. Click again to place the MLeader at desired location
  4. Repeat for multiple points or press Enter to exit

Interactive Options During Command:

  • Type S ‚Üí Change scale (default: 0.20)
  • Type A ‚Üí Change angle (default: 0.0)

Command Line Example:

Command: EN
Current scale: 0.20, Current angle: 0.0
Type S to change scale, A to change angle
Select point: [click point]
Place leader: [click location]
Coordinates: 12345.678, 98765.432
