üöÄ How to Use

  1. Type MO in the command line and press Enter
  2. Enter the offset distance
  3. Choose offset direction:
    • Outward (creates on "OFFSET-OUT" layer - Red)
    • Inward (creates on "OFFSET-IN" layer - Orange)
    • Both (creates both outward and inward)
  4. Select objects to offset
  5. View instant results with duplicate count

Command Line Example:
