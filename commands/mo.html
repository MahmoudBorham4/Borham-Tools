MO Command

Smart Multi-Offset - Copy objects inward/outward with specific distance and automatic duplicate prevention

üìå What is MO Command?

The MO (Multi-Offset) command is an enhanced and intelligent version of the Offset command in AutoCAD. It allows you to copy objects inward, outward, or in both directions with a specified distance, with automatic duplicate prevention and advanced layer/color management.

üéØ Key Features

  • 3 Copy Directions: Outward (Out), Inward (In), or Both directions
  • Smart Duplicate Prevention: Detects identical copies and automatically ignores them
  • Colored Layer Management: Red layer for outward, orange layer for inward
  • Batch Processing: Copy hundreds of objects at once
  • Live Progress Bar: Track created copies and ignored duplicates

  • When you need to offset many elements with the same distance
  • To create internal and external boundaries for buildings or lands
  • To draw double walls with specific thickness
  • To create road curbs and pathways
  • When you want to avoid duplicate copies automatically

üí° Tips and Best Practices

üéØ Choosing the Right Direction

  • Out: For external boundaries, external safety zones
  • In: For internal walls, building setbacks
  • Both: For roads with lanes on both sides, double walls

‚úÖ Verifying Results

  • Review final results message carefully
  • Check number of successful and duplicate copies
  • If there are failed copies, check the original elements
  • Use Layer Manager to control layers

‚ö†Ô∏è Avoiding Common Issues

  • Ensure layers are unlocked (Unlocked)
  • For small closed shapes: use distance smaller than half width
  • If inward offset fails, command will try outward automatically
  • Use Undo if you don't like the result
