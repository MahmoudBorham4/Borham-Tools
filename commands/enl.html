3D MLeader

ENL

Complete XYZ Coordinate Annotation with Elevation

üìñ Command Overview

The ENL command creates comprehensive MLeader annotations displaying full X, Y, and Z coordinates, perfect for 3D modeling and terrain analysis.

Key Features:

  • Complete 3D coordinates (X, Y, Z)
  • Shared settings with EN command
  • Automatic Z-coordinate detection
  • Professional multi-line MLeader format
  • Handles 2D points (Z = 0.0)
  • Interactive scale/angle adjustment

üöÄ How to Use

To annotate 3D coordinates in AutoCAD:

  1. Type ENL in the command line and press Enter
  2. Click on the 3D point you want to annotate
  3. Click again to place the MLeader at desired location
  4. Repeat for multiple points or press Enter to exit

Interactive Options During Command:

  • Type S ‚Üí Change scale (default: 0.20)
  • Type A ‚Üí Change angle (default: 0.0)

Command Line Example:

Command: ENL
Current scale: 0.20, Current angle: 0.0
Type S to change scale, A to change angle
Select 3D point: [click point]
Place leader: [click location]
Coordinates: 12345.678, 98765.432, 100.500

‚öôÔ∏è Shared Configuration

Global Variables (Shared with EN):

  • en_scale: Text scale (default: 0.20) - Shared with EN
  • en_angle: Text angle in degrees (default: 0.0) - Shared with EN

Layer Management:

Uses the same layer as EN command: "COORDINATES"

  • Color: Red (Index 1)
  • Consistency: Same layer for both 2D and 3D coordinates
  • Auto-unlock: Opens if locked or hidden
