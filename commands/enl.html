üí° Tips & best practices

‚úÖ For best results

  • Choose an appropriate Scale for the sheet size
  • Use OSNAP to snap precisely
  • Place labels away from geometry clashes
  • Keep consistent placement (e.g., all above or to the right)
  • Manage all labels via the COORDINATES layer

‚öôÔ∏è Adjust while working

  • Bigger text: type S then 0.30+
  • Smaller text: type S then 0.15-
  • Rotate: type A then set the angle
  • Exit: press Enter without picking a point
  • Last settings are remembered automatically

üé® Formatting tips

  • All labels go to the COORDINATES (red) layer
  • Change layer color in Layer Manager if needed
  • Use Properties to fine‚Äëtune after creation
  • Standard scale (0.20) suits most sheets
  • For A3 printing, try 0.25 or slightly higher
