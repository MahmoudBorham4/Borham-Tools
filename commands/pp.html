Command: PP

"Points on Polyline" - Create Surveying Points at Regular Intervals on Linear Entities

Purpose of the Command

The PP command is engineered to be the most precise and efficient tool for "staking out" points along a path in the AutoCAD environment. It accurately places AutoCAD Points on any linear entity—whether straight, curved, or closed—based on a user-defined interval.

Workflow

  1. Invoke Command: Type the shortcut PP into the Command Line and press Enter.
  2. Select Entities: You will be prompted to "Select objects:". Select all entities you wish to distribute points on.
  3. Enter Interval Distance: You will be prompted to "Enter interval distance:". Enter the numerical value for the desired distance between each point.
  4. Process Data: The command processes the entities and automatically places the points.

Advanced Technical Specifications

This command features several advanced technical capabilities that ensure accuracy and efficiency in a real-world work environment:

  • Duplicate Point Prevention:

    A key feature of this command is its automatic check for duplicate points. If the start point of one entity is the same as the end point of another, the command will place only a single point at that location, ensuring a clean drawing and accurate exported data.

  • Closed Entity Support:

    The command intelligently handles closed shapes. When a closed entity like a Circle or an Ellipse is selected, the command automatically places a single point at its geometric center, which is the required action in most surveying applications.

  • True Curve Measurement:

    The command ensures points are distributed along the true arc length of curves, not along the chords connecting the points. This feature is critical for maintaining high accuracy in projects containing arcs and complex curves.

  • Comprehensive Linear Entity Compatibility:

    The command has been tested to work with extreme precision on a wide range of 2D and 3D linear entities, including:

    • Line: Straight line segments.
    • Polyline (2D & 3D): Both 2D and 3D polylines.
    • Arc: Arcs of any radius.
    • Circle: Places a single point at the center.
    • Ellipse: Places a single point at the center.
    • Spline: Complex, free-form curves.

Advanced Use Case

Problem: You have a plot plan that includes straight boundaries (Lines), curved roads (Arcs), and a roundabout (Circle). You need to place stakes every 10 meters along the entire perimeter, plus identify the center of the roundabout.

Solution with PP:

  1. Run the PP command.
  2. Select all the lines, arcs, and the circle that form the plan in a single selection.
  3. When prompted for the distance, enter 10.
  4. Instant Result:
    • Points will be accurately distributed every 10 meters along the lines and arcs.
    • A single point will be placed at the center of the roundabout.
    • No duplicate points will be created where the lines and arcs connect.
