Command: PP

"Points on Polyline" - Create points along linear objects.

Description

The PP command creates AutoCAD Points along selected linear objects (like lines, polylines, arcs, and circles) at a regular, user-defined interval. This is extremely useful for layout work and dividing distances in surveying and road projects.

How to Use

  1. Run Command: Type PP in the command line and press Enter.
  2. Select Objects: AutoCAD will prompt you to "Select objects:". Choose all the linear objects you want to place points on.
  3. Set Interval: You will be prompted to "Enter interval distance:". Enter the desired distance between each point.
  4. Result: The command will automatically place points along the selected objects at the specified interval.
