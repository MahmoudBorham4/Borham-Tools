Command: PP

"Points on Polyline" - Create Surveying Points at Regular Intervals on Linear Entities

Purpose of the Command

The PP command is designed to automate the process of "staking out" points along a path within the AutoCAD environment. It accurately places AutoCAD Points on any linear entity, whether straight or curved, based on a user-defined interval.

This command is an indispensable tool for surveyors and civil engineers when dealing with road layouts, utility lines (power, water, sewage), or any project that requires precise distance division.

Workflow

  1. Invoke Command:

    Type the shortcut PP into the Command Line and press Enter.

  2. Select Entities:

    You will be prompted to "Select objects:". Select all the linear entities on which you want to distribute the points. You can select multiple different objects in a single operation.

  3. Enter Interval Distance:

    You will be prompted to "Enter interval distance:". Enter the numerical value for the desired distance between each point. The distance is calculated along the true length of the entity.

  4. Process Data:

    The command processes the selected entities and automatically places the points. The points will appear directly on the drawing, typically on a dedicated layer.

Technical Specs & Features

  • Comprehensive Entity Support: Works efficiently on Lines, Polylines, Arcs, Circles, and Splines.
  • Curve Accuracy: Ensures points are distributed correctly along the true length of curves and arcs, not along their chords, providing high surveying accuracy.
  • Batch Processing: You can select an unlimited number of objects at once, and the command will process them all in a single operation.
  • Performance Efficiency: The algorithm is optimized to run quickly, even on large and complex drawings.

Use Case

Problem: You have a 2.5 km pipeline route drawn as a Polyline, and you need to mark the locations of welding joints every 12 meters along the path.

Solution with PP:

  1. Run the PP command.
  2. Select the Polyline representing the pipeline.
  3. When prompted for the distance, enter 12.
  4. Result: AutoCAD points will be instantly created along the path exactly every 12 meters, ready to be exported to a Total Station or used for generating drawings.
