The PP Command is an intelligent AutoCAD tool that automatically generates precise geometric points on selected elements, eliminating hours of manual work.

  • Creates points at the center of circles
  • Generates points at the vertices of polygons
  • Divides arcs and curves into evenly spaced points
  • Processes multiple element types simultaneously

üöÄ How to Use the PP Command

Using PP Command takes just 4 simple steps:

1

Start the Command

Type PP in AutoCAD command line and press Enter

2

Enter Spacing Value

Enter point spacing for curves (or press Enter for default 0.30 meters)

3

Select Elements

Select all circles, polygons, and arcs you want points on (you can select hundreds at once)

4

Done!

Watch points generate automatically in under a second with point count displayed

üí° Quick Tips:
